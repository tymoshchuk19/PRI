Abstract

Nesta etapa é proposto o desenvolvimento de um servidor em node js utilizando o framework express e garantindo a presistência em mongodb, com o propósito de geririr um data set, onde podem ser adicionados e removidos filmes.

Foi então necessário criar um ficheiro de rotas filmes.js que trata de interpretar o pedido, por uma questão de gestão foram tambem criadas rotas de api, sendo estas a tratar da comunicação com a base de dados. Para a ligação á base de dados é nessesário a criação de um modelo, filme, e para efetuar operações utilizamos um controlador sobre o modelo.

Foi também criado um script, adicional ao front-end, para efetuar chamadas através da libraria axios, no caso em questão para efetuar a chamada há rota de remoção de um filme. Para além dos ficheiros em pug geradores das paginas html que permitem a intereação co o utilizador.
