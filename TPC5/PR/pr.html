Abstract

Nesta etapa é proposto o desenvolvimento de um servidor em node js utilizando o framework express , com o propósito de funcionar como uma espécie de pauta, onde podem ser adicionados e removidos alunos e efetuar as mesmas operações para as respetivas notas de cada aluno.

Foi então necessário a criação de um ficheiro de rotas alunos.js que trata de interpretar o pedido, efetuar as alterções necessárias á informação e responder com a página atualizada. Foram também criados ficheiros adicionais para efetuar chamadas através da libraria axios, para além dos ficheiros em pug geradores das paginas html que permitem a intereação co o utilizador.
