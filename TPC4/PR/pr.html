Abstract

Nesta etapa é proposto o desenvolvimento de um pequeno servidor em node js que responde ao pedido de cada musica (rota: musica/num), com o respetivo ficheiro xml.

Para tal, foi necessário processar o documento com todas as musicas de forma a obter documentos individuais, como é o exemplo do doc1.xml. O servidor precisa de respoder ás rotas que devolvem musicas obtendo o ficheiro xml individual que é carregado pelo browser e consequentemente manda um pedido para a obter o ficheiro doc2html.xml que permite a rederização de cada ficheiro xml.
