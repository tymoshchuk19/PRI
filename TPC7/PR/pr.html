Abstract

Nesta etapa é proposto o desenvolvimento de um servidor em node js utilizando o framework express e garantindo a presistência em mongodb, com o propósito de geririr um data set, onde podem consultadas obras e os compositores com mais obras.

Foi então necessário criar um ficheiro de rotas api.js que trata de interpretar o pedido. Para a ligação á base de dados é nessesário a criação de um modelo, obras, e para efetuar operações utilizamos um controlador sobre o modelo.
